# 🏥 Hospital Bed Management System

A comprehensive C-based application for managing hospital ward operations, bed allocation, patient admissions, transfers, and discharges with role-based access control.

## ✨ Features

### 🔐 Authentication System
- Role-based access (Admin/Staff)
- User registration with admin approval
- Secure login system

### 🏥 Ward Management
- Add/Remove wards
- Track ward capacity
- Monitor bed availability

### 🛏️ Bed Management
- Add/Remove beds
- Real-time bed status (Available/Occupied/Maintenance)
- Automatic bed allocation

### 👨‍⚕️ Patient Management
- Patient admission with bed assignment
- Transfer patients between beds/wards
- Patient discharge with bed release
- Complete patient history tracking

### 📊 Reporting System
- Bed occupancy reports
- Ward statistics
- Patient reports
- Export to file functionality
